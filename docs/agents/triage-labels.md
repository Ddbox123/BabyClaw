# Triage Labels

This repository uses the default Matt Pocock triage label vocabulary.

| Semantic role | GitHub label |
| --- | --- |
| Needs triage | `needs-triage` |
| Needs more information | `needs-info` |
| Ready for agent work | `ready-for-agent` |
| Ready for human work | `ready-for-human` |
| Won't fix | `wontfix` |

Agents should preserve these meanings when triaging or updating GitHub Issues.
